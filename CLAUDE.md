# Purfroller — Project Instructions

## CAD Tooling

Use the **build123d MCP server** by default for all CAD work:
- `mcp__build123-mcp__execute` — run model code
- `mcp__build123-mcp__render_view` — visualise
- `mcp__build123-mcp__measure` — check dimensions
- `mcp__build123-mcp__export` — produce STL

Edit `purfroller.py` only to persist finalised changes.

## Process

**Talk before coding.** For any design change, discuss intent and approach first. Ask about constraints, tradeoffs, and assembly implications before touching any file.
